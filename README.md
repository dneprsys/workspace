# ⚙️ CNC Мониторинг Pro

Мобильное приложение для мониторинга и управления CNC-станками.  
Построено на **React Native** + **Expo**.

## 📱 Возможности

- ⏱ **Таймеры станков** — запуск, пауза, сброс, обратный отсчёт с прогресс-баром
- 📊 **Расчёт смен** — автоматический подсчёт деталей за смену, эффективность, остаток
- 📏 **Расчёт заготовок** — деталей из заготовки, остаток, время замены
- 📱 **Telegram-уведомления** — предупреждение за N минут, уведомление о завершении
- 🔔 **Push-уведомления** — через Expo Notifications
- 🧮 **Встроенный калькулятор** — для быстрых расчётов прямо в приложении
- 📤 **Экспорт истории** — в CSV и JSON
- 💾 **Резервное копирование** — сохранение и восстановление настроек

## 🚀 Запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npx expo start

# Android
npx expo start --android

# iOS
npx expo start --ios
```

## 📁 Структура проекта

```
├── App.js                          # Главный компонент
├── src/
│   ├── components/
│   │   ├── CalcModal.js            # Модальный калькулятор
│   │   ├── HistoryPanel.js         # Панель истории завершений
│   │   ├── MachineCard.js          # Карточка станка (таймер/смена/заготовка)
│   │   └── TelegramPanel.js        # Настройки Telegram
│   ├── hooks/
│   │   └── usePersistence.js       # Хук отложенного сохранения
│   ├── services/
│   │   ├── mathParser.js           # Безопасный парсер мат. выражений
│   │   ├── notifications.js        # Push-уведомления (Expo)
│   │   ├── reports.js              # Генерация CSV/JSON отчётов
│   │   ├── storage.js              # AsyncStorage CRUD
│   │   └── telegram.js             # Telegram Bot API
│   ├── utils/
│   │   └── machine.js              # Фабрика станков, форматирование, расчёты
│   └── theme.js                    # Цвета, стили, пресеты
├── app.json                        # Конфигурация Expo
└── package.json
```
